How to include jsfit

jsfit makes heavy use of the numeric.js library, which must be included before including the jsfit file.

Check out the examples on this page, but here is the basic recipe for using jsfit.

//define a function that we will use as the model to minimize //The function must accept a single x value, and an array of parameters function linear (x, pars) { var m = pars[0]; var b = pars[1]; return m * x + b; } //the indenpendent values var xvals = [...]; //the dependent values var yvals = [...]; //this is what we pass into the fitter var data [ xvals, yvals]; //p0 are the initial parameters var p0 = [1.0, 10.0]; //the fitobj is the results of the fitting var fitobj = jsfit.fit(linear, data, p0);
